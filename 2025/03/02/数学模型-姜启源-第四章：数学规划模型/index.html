第四章：数学规划模型

要点摘录

  • 线性规划模型
    • 使用 LINGO 来求解
    • 紧约束
    • 影子价格
    • 对目标函数系数的敏感性分析
    • 对资源约束右端项的敏感性分析
  • 整数规划求解
  • 处理模型中的分段函数
    • 为每个区间设一个决策变量，通过决策变量间的数量关系来表示分段函数，可能化为非线性规划
    • 引入 0-1 变量，为每个区间设一个决策变量，通过决策变量间的数量关系来表示分段函数，化为线性规划
    • 直接处理分段线性函数，原理未理解，记住如何使用（P111）
  • 题干中的对象之间有先后关系 => 将中间物也看作一个对象，为其设置相关决策变量，以此来连接两端的对象之间的关系
  • 指派问题（引入 0-1 变量来建模）
  • 单目标规划与多目标规划（解决方法是：通过加权组合形成一个新的单一目标）
  • 生产批量问题（还是引入 0-1 变量，0-1 变量有强大的表达力：表示选择）
  • 生产下料问题
  • 目标规划
  • 现代投资组合理论
  • 均值-方差模型
  • 分离定理
